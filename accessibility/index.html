Accessibility

Built to work for more people

Use this website with a keyboard, screen reader, zoom, reduced motion, or your preferred contrast and color settings.

What is supported

  • Clear headings, landmarks, page titles, skip links, visible keyboard focus, and no pointer-only actions.
  • Responsive layouts that reflow without horizontal page scrolling.
  • Auto, Light, Dark, Soft, and High contrast appearances, alongside system accessibility preferences.
  • Keyboard and screen-reader support in Ask Mantosh, including focus management and completion announcements.

Known limits

  • The résumé PDF is not currently tagged; use the HTML résumé or experience page instead.
  • English is the authoritative version. Browser translation can be used without sending content through a website translation widget.
