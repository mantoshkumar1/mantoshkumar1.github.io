# Engineering Work Outside Nokia

This document keeps Mantosh Kumar's non-Nokia experience attributable to the correct employer. It is derived from the reviewed public résumé and supports direct questions about work performed before or outside Nokia.

## Role-by-role evidence

- **KI Labs GmbH:** Built Python and Django backend services, REST APIs, PostgreSQL data models, and business workflows for customer onboarding, regulatory compliance, customer verification, and risk-based order validation.
- **Siemens AG:** Built SDN/NFV research prototypes for network monitoring, intent specification, intent-driven orchestration, and quality-of-service validation using OpenDaylight, Mininet, RESTCONF, and YANG.
- **Intel Corporation:** Built cellular-message and modem-performance analysis tools, the initial version of a unit-level modem-validation framework, 3GPP-derived validation automation, and project-status reporting automation.
- **Cisco Systems:** Built Python packet-core simulation tools and telecom feature-validation automation across 2G, 3G, LTE, Wi-Fi, and eHRPD technologies.
- **Aricent Group:** Integrated Facebook and location-based services for a unified social-networking prototype and designed platform-integration use cases and workflows.
